# Everest Gym Website

Static modern gym website.

## Deploy on GitHub Pages
1. Upload all files to a public GitHub repository.
2. Go to Settings > Pages.
3. Source: Deploy from branch.
4. Branch: main, folder: /root.
5. Save and open the GitHub Pages URL.

## Important
Replace `your-email@example.com` in index.html with the gym owner's email if using FormSubmit.
